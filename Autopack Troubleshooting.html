The autopack is not responding, at all

1)      Make sure the layon button is pressed on the winder. If not, press it.

2)      Check the autopack E-stop.

3)      Check if power is connected

a.       Check the main power switch

b.       Check the Power socket is connected

4)      Check that the sockets and air are connected to the autopack

5)      Try closing and opening the red air stop valve.

6)      Make sure the autopack is lowered. Check the lifting actuator sensors on all the feet of the autopack. If any of the sensors are active, move them down so that they deactivate.

7)      Open the panel to see if either linear actuator is in error. This will be indicated by a red LED on the controller. If there is an error, follow the Linear Actuator Troubleshooting Procedure

8)      Check the 3 position switch on the PLC and make sure it is set to run

9)      Check the main fuse for the autopack.

10)   The colored dot on the autopack must match the colored dot on the winder. Otherwise the autopack will not work.

The linear actuators move, but there is no air pressure

1)      Make sure the valve is open on the winder and that the air hose is connected to the correct port.

2)      Try closing and opening the red air stop valve.

3)      Make sure the main pressure regulator is fully open (about 90psi). Try closing and opening the red air stop valve.

4)      Listen for air leaks in the regulator manifold. If this was the case, it should be a very loud.

5)      On the bottom of the red air stop valve is a flow controller screw. On new air stops this is closed. Back out the screw to enable air flow. Actuate the red air stop valve.

6)      Is the orange light on for the air stop? If not, open the panel to see if either linear actuator is in error. This will be indicated by a red LED on the controller. If there is an error, follow the Linear Actuator Troubleshooting Procedure.

7)      The colored dot on the autopack must match the colored dot on the winder. Otherwise the autopack will not work.

Air pressure is present but the table pusher is not pushing the box from the LOADING position.

1)      Make sure there are no boxes in the way.

2)      Reset the autopack. An internal error may have occurred. Press and release the E-stop.

3)      Check if the diffuse sensor on the middle of the table is reading its own reflection off the plexiglass. If it is, place rubber pad over the area illuminated on the plexiglass to cover the reflection.

4)      Did the loading position sensor light come on?

a.       Make sure the autopack has power. Check if the other sensor lights come on.

b.       Make sure the connector is secure.

c.       Adjust the sensitivity dial on the sensor (gently) to increase the detection range.

d.       If adjustments dont activate the sensor, replace it.

5)      Are the middle dividers up?

a.       Check that the led on the sensor on the divider actuator is illuminated for both dividers. If not, adjust the position of the sensor. If the sensor is never triggered, replace the sensor

6)      On the home side for the table slide actuator, check if the sensor LED is triggering. If it is not, back off the cylinder adjustment for the air cushion (on the end of the rail), then move the position of the sensor.

7)      Try adjusting the rotary actuator sensor position.

Air pressure is present but the table pusher is not pushing the box from the MIDDLE position.

1)      Is the case complete? If not, the autopack will not release the case.

2)      Make sure the air pressure is on for the autopack.

3)      If the table is sticky from tape residue, clean the table.

4)      Is the sensor on the exit of the table slide activated?

a.       Clear what is blocking the end of the table.

b.       Adjust the sensitivity of the sensor (gently) to not trigger a false positive.

5)      Make sure both middle divider sensors are activated. If not, adjust the sensor position and/or divider up pressure

6)      Is the divider crooked? This could case the roll to be wedged. Replace the divider.

7)      Check the side rails

a.       If the side rails are too tight, the box wont pass.

b.       If the side rails are adjusted too far to one side, the box will catch on one of the escapement cylinders.

8)      Reset the autopack It could be an internal error.

9)      Try increasing the pressure on the table slide pressure regulator.

The gripper will not move to grab the roll.

1)      If the green strobe light is not on, then follow the troubleshooting steps for The autopack is not responding, at all

2)      Is there a box in the middle position? The autopack will not grab without a box loaded.

3)      Is the box in the middle position complete? Follow the guide for Air pressure is present but the table pusher is not pushing the box from the MIDDLE position.

4)      Make sure the connection to the ultrasonic sensors are secure. Check that the LED indicators are on. If not, adjust the connection. If they do not turn on with a good connection, replace the sensor and program the parameters.

5)      Clear any film, tape or glue blocking the ultrasonic sensor from reading the roll size. Sensor is located under the winder drop plate.

6)      Make sure the roll size I correct. If not, have the operator fix it.

7)      Make sure the film is centered on the core. If it is not centered, the ultrasonic sensor may not read.

8)      Check for an error on the linear rail controller. If so, follow the Linear Actuator Troubleshooting Procedure

9)      If the light is not on the linear actuator controller, check the fuses for the controllers.

10)   Teach the autopack the correct size by turning the QC switch up 3 times within 2 seconds. The autopack will need 3 good readings on the ultrasonic sensors to calibrate. Bad readings are rejected. The green strobe flashes slow for a good reading. A maximum of 6 readings are taken. If there have been 6 roll changes and still the problem persists, continue with the other troubleshooting steps.

11)   For large roll, make sure divider down sensor is active, and adjust the position if not. If the sensor never triggers, replace it.

12)   Follow the Ultrasonic Troubleshooting Procedure

13)   Make sure the analog module is fully inserted into the PLC. When fully inserted, the clips on the top and bottom will click in place as you push them. Make sure the wire connector is attached to the module.

14)   If the ultrasonic sensors are reading well (check this first), then replace the analog module, K#15518, on the PLC.

The gripper moves to grab the roll, but does not close or drops the roll.

1)      If the green strobe light is not on, then follow the troubleshooting steps for The autopack is not responding, at all

2)      Is there a box in the middle position? The autopack will not grab without a box loaded.

3)      Is the box in the middle position complete? Follow the guide for Air pressure is present but the table pusher is not pushing the box from the MIDDLE position.

4)      Make sure the film is centered with the core.

5)      If the divider is bent too much, it could cause the roll to not drop into the box. This is more so an issue for mini rolls. Replace the divider as needed.

6)      For the mini 100 and mini 300 autopacks, is the gripper pad installed and clean? Replace the pad if necessary. Sandpaper, tape or double stacked pads are not acceptable.

7)      For large roll, make sure divider down sensor is active, and adjust the position if not. If the sensor never triggers, replace it.

8)      Make sure the gripper pressure is the maximum. If not, increase the pressure.

9)      If the core is not flush against the finger holder, adjust the position of the core loader chute to compensate.

10)   Does the core spin (ie, move) on the chucks?

a.       Make sure the center pivot is not damaged and the tips of the chucks do not show wear. For mini roll, the center pivot is K#6030. For large roll the center pivot is K#6026.

b.       Make sure the chucks are not bent.

11)   If the layon rollers are worn, replace them. This will help with the roll tension.

12)   For large roll, make sure the gripper is hitting the roll. If not, follow the Linear Actuator Troubleshooting Procedure

13)   If the chuck speed is slow to retract, open the flow controller on the chuck cylinder.

14)   If the chuck speed is too fast, increase the front chuck delay setting in the HMI.

15)   If the roll is wound too tight it may be holding on to the finger. Reduce the tension on the winder.

16)   If the box is obtusely tilted,

a.       Adjust the side rails

b.       Lower the speed of the box pusher.

Ultrasonic Troubleshooting Procedure

1)      Use maintenance laptop to download parameters to ultrasonic sensors. Pictured is the cable used for programming.

2)      Connect the cable USB pin to the USB plug in the back side of the laptop as shown in the image

3)      Connect any one (doesnt matter which one) of the orange cables from the autopack to the T-adapter as shown in the image.

Make sure that the autopack is powered on.

4)      Connect the Orange cable, which is connected to the laptop USB cable to the Ultrasonic sensor in the winder. Note that some of the lines have an extension cable for convenience

5)      Open SICK Connect+ from Start Menu.

 

6)      In the upper left hand corner window, follow Setup -> Search for Connect+ adapter click

 

7)      Disconnect the adapter, then click the red button on the screen

 

8)      Reconnect the adapter, and click the yellow button.

 

9)      The software will find the COM port automatically.

10)   Click on the read parameter from sensor button, or press 1 on the keyboard

11)   If asked to press the reset button, press and hold the R button on the adapter

12)   If prompted, read the sensor parameter

13)   The window should show the parameters currently downloaded to the sensor.

14)   The correct parameters are as follows:

a.       Mini 100: 204mm to 286mm

b.       Mini 300: 30mm to 250mm

c.       Large: 150mm to 230mm

15)   To adjust a parameter, click on the value, type the new value and hit ENTER

 

16)   Download the new parameters to the sensor by clicking on the icon as shown in the image.

17)  Click Yes

18)   To program the other sensor, remove the orange cable from the 1st ultrasonic sensor in the winder and plug it to the other ultrasonic sensor in the winder. Repeat exact same procedure for the other sensor.

19)   To see the sensor reading, click on Measure on the top of the window, then click measurement writer.

20)   A screen will come up showing the sensor reading.

21)   The reading should be a gently sloped line as shown. If the reading is completely horizontal, or if the reading has spike values, then the sensor is not reading well, and either needs to be adjusted or replaced.

22)   If the sensor is not reading well, clean it and check it for damage. If damaged, replace it.

23)   Check if the bracket has come loose or if the sensor has become misaligned in some way. Adjust the bracket and sensor position until the reading is a gently sloped line, then lock the adjustment screws to hold the position. The adjustment does not need to be perfect as long as it is reading. Teaching the autopack will correct minor differences in positioning.

 

Linear Actuator Troubleshooting Procedure

This SOP is to use when adjustments are needed to the positions of the linear actuator rails

1.       Use cable shown in the image below to download required step data to linear actuator controller. For the older controllers use the LEC adapter. For new controllers use the JXC adapter.

2.       Connect the one end of the cable to the USB port on left side as shown in the image below.

3.       Connect to the linear actuator controller. Make sure the controller is powered on.

 

4.       Open ACT controller (LE icon at the bottom of the screen)

Click on Normal mode in newly opened window.

5.       Window shown in the image below should appear on the screen.

6.       Select the correct COM port.

Go to Action -> System -> Settings

7.       Type Device Manager into the Windows search bar and click on the top result to open the Device Manager.

8.       In the Device manager, check the COM port that the actuator is connected to. The port will read SMC Serial Port

9.       Switch back to the LE ACT Controller software and make sure the COM port matches that in the Windows Device Manager or change it to match by clicking the port number and selecting the correct port from the drop-down.

Then click OK

 

10.   Reset the software. Go to Action -> System -> Reset

11.   If the drive is in error, the Alarm button will be flashing red. Click the alarm to see the issue. For any issue regarding over voltage or over current, replace the controller.

12.   STEP data window is shown in the image below.

If the window doesnt open, click on VIEW > STEP DATA

13.   Click on Upload LE -> PC

14.   The current step data will load on to the laptop. If it is empty, ie. if the controller is new, then Upload the step data from a known working controller. Conveniently, each autopack has two controllers, so the step data can be taken from that controller.

15.   Monitor the step that the actuator is at by opening the status window. Click View -> Status


16.   In the window, identify the step or steps that need their position adjusted. For reference, step 1 is always the waiting position, and step 2 is always the grabbing position. A step may need to be adjusted if the rolls are being dropped in the wrong location, or if the autopack is not going all the way to grab the roll.

 

17.   Return to the Step Data window and adjust the values in the position column as needed. The values shown are the distance from the back of the actuator. Changed values are shown in blue. Impossible values are shown in red. Make sure to only change the values in the position column. Be careful of using values close to the limits (Limits are 0&900mm mini roll, 0&1000mm large roll). The rails have inertia, and if it exceeds the maximum set positions, it will throw an error. DO NOT adjust the pushing or trigger values.

18.   Once the values are set, click on Download PC>LE to send parameters to the controller

19.   Unplug the cable from the controller and plug it into the second controller, then click on Download PC>LE to send parameters to the controller
